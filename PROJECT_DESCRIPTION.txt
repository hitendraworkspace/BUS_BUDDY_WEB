WHAT IS BUS BUDDY?
================================================================================

Bus Buddy is a comprehensive school bus tracking and fleet management platform
designed to ensure student safety and streamline school transportation
operations. It provides real-time GPS tracking, safety monitoring, and
management tools for schools, parents, drivers, and administrators.

The system helps schools:
- Track buses in real-time
- Monitor driver behavior and safety
- Manage routes efficiently
- Generate reports and analytics
- Ensure student safety during transportation

HOW IT WORKS
================================================================================

1. HARDWARE INTEGRATION
   - GPS tracking devices installed in buses
   - MDVR cameras for video monitoring
   - DMS/ADAS sensors for driver monitoring
   - All devices connected to central server

2. DATA COLLECTION
   - GPS devices send location data every few seconds
   - Cameras record video continuously
   - Sensors detect driver behavior and safety events
   - All data transmitted to cloud server

3. DATA PROCESSING
   - Python scripts process alarm data from tracking server
   - Alarms are grouped and categorized
   - Data stored in Supabase database
   - Real-time updates pushed to web application

4. WEB APPLICATION
   - Administrators log in to dashboard
   - View real-time bus locations on map
   - Monitor safety alarms and alerts
   - Manage routes and drivers
   - Generate reports

5. NOTIFICATIONS
   - Real-time alerts for safety events
   - Route status updates
   - Driver behavior notifications
   - System status updates
